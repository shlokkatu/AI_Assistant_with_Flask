# 🤖 AI Assistant with Flask

A simple **AI Personal Assistant Web App** built using **Flask**.  
It provides two main features:
- **Ask Anything** – Get instant answers to your questions.  
- **Summarize Email** – Paste an email and receive a concise summary.  

---

## ✨ Features

- Interactive web interface with clean design (HTML + CSS).  
- Flask backend with two endpoints:
  - `/ask` → Handles user questions.  
  - `/summarize` → Summarizes email content.  
- Responsive UI with loading indicators.  
- Easy to extend with any AI model or API integration.  

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)  
- **Frontend:** HTML, CSS, JavaScript (Fetch API)  
- **Styling:** Custom CSS (modern, responsive design)  
- **Deployment:** Works locally and can be deployed on Heroku, Vercel, or Docker  

---

📖 Usage
Open the app in your browser.

Use the Ask Anything form to type a question → get instant AI response.

Use the Summarize Email form to paste an email → get a concise summary.
